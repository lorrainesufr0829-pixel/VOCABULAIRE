算法伪代码\n\n```python\ndef calculateNextReview(record, isCorrect, difficulty):\n    if not isCorrect:\n        # 答错：重置复习间隔\n        interval = 1\n        easeFactor = max(1.3, easeFactor - 0.2)\n    else:\n        # 答对：按规则递推\n        if interval == 0:\n            interval = 1\n        elif interval == 1:\n            interval = 3\n        else:\n            interval = round(interval * easeFactor)\n        \n        # 更新难度系数\n        # difficulty 范围：0-5\n        # 0 = 太简单，5 = 太难，3 = 中等\n        easeFactor = max(\n            1.3,\n            easeFactor + (0.1 - (5 - difficulty) * 0.08)\n        )\n    \n    nextReviewDate = today + interval days\n    return {interval, easeFactor, nextReviewDate}\n```\n\n### 参数说明\n\n- **interval**: 两次复习之间的天数\n  - 初始值: 0\n  - 递推: 1 → 3 → 7 → 14 → 30 → 60 → 180 ...\n\n- **easeFactor**: 难度系数\n  - 初始值: 2.5\n  - 范围: >= 1.3\n  - 越大: 复习间隔越长\n  - 越小: 复习间隔越短\n\n- **difficulty**: 用户的难度评分\n  - 0-2: 太简单/简单，提高难度系数\n  - 3: 中等，难度系数保持稳定\n  - 4-5: 有点难/太难，降低难度系数\n\n## 🛠️ 常见开发任务\n\n### 添加新的复习模式\n\n1. 创建新组件 `src/components/ReviewModeXXX.vue`\n2. 在 `Review.vue` 中添加模式选项\n3. 处理复习结果和记录\n\n### 集成发音功能\n\n```typescript\n// src/utils/index.ts\nexport
