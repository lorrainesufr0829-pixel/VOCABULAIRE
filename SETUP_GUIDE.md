# 🚀 项目设置指南\n\n欢迎使用 **法语词汇小卖部**！这份指南会帮助你快速上手。\n\n## 📋 环境要求\n\n- Node.js >= 16.0\n- npm >= 8.0 或 yarn >= 3.0\n- 现代浏览器（Chrome, Firefox, Safari, Edge）\n\n## 🎯 快速开始 (5分钟)\n\n### 1️⃣ 克隆或下载项目\n\n```bash\ncd VOCABULAIRE\n```\n\n### 2️⃣ 安装依赖\n\n```bash\nnpm install\n```\n\n或使用 yarn:\n\n```bash\nyarn install\n```\n\n### 3️⃣ 启动开发服务器\n\n```bash\nnpm run dev\n```\n\n浏览器会自动打开 `http://localhost:5173`\n\n### 4️⃣ 开始使用！\n\n🎉 应用已成功启动！\n\n## 📚 功能速览\n\n### 首页 (Home)\n- 查看今日学习进度\n- 掌握率概览\n- 学习建议\n\n### 单词库 (Words)\n- ➕ 添加单个单词\n- 📤 导入 CSV 文件\n- 📥 导出单词\n- 🔍 搜索和分类筛选\n\n### 复习中心 (Review) ⭐ 核心功能\n- 🎴 卡片翻转模式\n- 🎯 选择题模式\n- ⚡ 快速复习模式\n- 📊 复习统计\n\n### 统计分析 (Stats)\n- 📈 学习进度图表\n- 🔥 困难词汇排行\n- 📅 复习热力图\n\n### 设置 (Settings)\n- 🎨 主题切换\n- 🎯 学习目标设置\n- 💾 数据备份/恢复\n\n## 📖 使用教程\n\n### 添加你的第一个法语单词\n\n1. 点击导航栏的 \"📚 单词库\"\n2. 在「快速添加」区域输入：\n   - 法语: `bonjour`\n   - 中文: `你好`\n   - 分类: `问候`\n3. 点击 \"➕ 添加\" 按钮\n4. 看到成功提示后，单词已保存！\n\n### 批量导入单词 (CSV)\n\n#### CSV 文件格式\n\n创建 `words.csv` 文件：\n\n```csv\nbonjour,你好,问候\nmerci,谢谢,礼貌\ns'il vous plaît,请,礼貌\noui,是,基础\nnon,否,基础\neau,水,名词\npain,面包,名词\n```\n\n然后在「数据管理」区域点击 \"📤 导入 CSV\" 选择该文件。\n\n### 开始复习\n\n1. 点击导航栏的 \"✍️ 复习\"\n2. 看到「选择复习模式」卡片，选择你喜欢的模式：\n   - **卡片模式**: 点击卡片翻转查看中文\n   - **选择题模式**: 从4个选项中选择正确答案\n   - **快速复习**: 快速浏览所有单词\n3. 根据难度反馈：\n   - 🟢 太简单/简单/有点简单\n   - 🟡 中等\n   - 🔴 有点难/太难\n4. 系统会根据你的反馈自动安排下次复习时间\n\n### 查看学习统计\n\n1. 点击导航栏的 \"📊 统计\"\n2. 查看：\n   - 词汇掌握分布\n   - 最近7天复习记录\n   - 困难词汇排行榜\n\n## 🧠 记忆曲线工作原理\n\n这个应用使用 **SM-2 艾宾浩斯记忆曲线**：\n\n```\n第1次复习: 学习后立即\n第2次复习: 1天后\n第3次复习: 3天后\n第4次复习: 7天后\n第5次复习: 14天后\n第6次复习: 30天后\n第7次复习: 60天后\n第8次复习: 180天后\n```\n\n**关键机制**:\n- ✅ **答对**: 间隔延长，难度系数提高\n- ❌ **答错**: 间隔重置为1天，难度系数降低\n- 📊 **难度评分**: 你的反馈（0-5分）会调整复习间隔\n\n## 💾 数据存储说明\n\n### 本地存储\n\n所有数据保存在浏览器的 **LocalStorage** 中：\n\n- `vocab_words`: 单词数据\n- `vocab_records`: 学习记录\n- `vocab_history`: 复习历史\n\n**优点**: 无需服务器，完全离线使用\n\n**注意**: 清除浏览器缓存会丢失数据\n\n### 备份和恢复\n\n#### 导出备份\n\n1. 进入 \"⚙️ 设置\"\n2. 点击 \"📥 本地备份\"\n3. 浏览器会下载 `vocab_backup_*.json` 文件\n\n#### 导入备份\n\n1. 进入 \"⚙️ 设置\"\n2. 点击 \"📤 导入备份\"\n3. 选择之前导出的 JSON 文件\n4. 数据会自动恢复\n\n## 🎨 自定义配置\n\n### 修改主题颜色\n\n编辑 `src/components/Mascot.vue` 中的 `messages` 数组。\n\n## 🔧 开发工具\n\n### 代码检查\n\n```bash\nnpm run typecheck\n```\n\n### 生产构建\n\n```bash\nnpm run build\n```\n\n生成的文件在 `dist/` 目录\n\n### 预览生产版本\n\n```bash\nnpm run preview\n```\n\n## 📱 项目大小\n\n- 源代码: ~200KB\n- 构建后: ~150KB (压缩)\n- 极轻量，秒速加载\n\n## 🐛 常见问题\n\n### Q: 数据会丢失吗？\n**A**: 如果你清除浏览器缓存，所有数据会丢失。记得定期备份！\n\n### Q: 可以在多个设备上同步吗？\n**A**: 当前版本不支持自动同步。可以通过导出/导入
